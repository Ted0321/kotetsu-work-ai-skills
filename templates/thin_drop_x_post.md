# 薄配布用 X投稿テンプレ

```text
【フック1行】
（損・あるある・禁止・数字）

【何が手に入るか1行】
（コピペで何ができるか）

【本体】
（プロンプト or 手順 3〜7個）

【使い方3行】
1.
2.
3.

【締め】
保存用 / 自分の案件で使うなら「〇〇」を入れ替えて
```

## 例（論点ミニ版）

```text
AIに相談してるのに、会議が決まらない人へ。

原因の9割は、論点が切れてないこと。

コピペ用：論点を「決めること」に変える3問

1. 今日、Yes/Noしたいことは何？
2. それを決めるために必要な論点は3つだけ言うと？
3. 今やらなくていい論点は何？（捨てる）

使い方
1. この3問を案件名と一緒にAIへ貼る
2. 出てきた論点を会議タイトルにする
3. 資料はその後

保存して、資料作成前に使って。
```
