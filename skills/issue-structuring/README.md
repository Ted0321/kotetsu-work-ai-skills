# 論点整理スキル（issue-structuring）

雑な相談・会議メモ・ふわっとした依頼を、**意思決定できる論点ツリー**に整理します。

- 資料を書き始める前
- 方針会議の前
- AIに深掘りさせる前

に使ってください。

## クイックスタート

1. [SKILL.md](./SKILL.md) を開く  
2. 全文を Claude / ChatGPT / Gemini に貼る  
3. 末尾に案件メモを書く  

入力例と理想出力は [examples/sample_input_output.md](./examples/sample_input_output.md)

## 出力されるもの

1. Decision Ask（決めること・1文）
2. 論点ツリー
3. 今やる Top3
4. 今やらない論点
5. 足りない事実
6. 次アクション
7. 次にAIへ投げるプロンプト

## 注意

- 解決策の提案が目的ではありません
- 論点を増やして賢く見せるためのものでもありません
- 「切る」「捨てる」「決める」ためのスキルです
