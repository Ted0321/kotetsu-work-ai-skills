# 調査から示唆への変換例

## 調査結果

- 競合3社が同じ新機能を半年以内に追加した
- 利用者インタビューでは、その機能より導入時の設定負荷が不満として多い
- 自社の解約理由でも初月離脱が目立つ

## 弱い出力

```text
市場では新機能への注目が高まっている。自社も機能強化を検討すべきである。
```

## 良い示唆

- なぜなら: 競合は機能を追加している一方、顧客の離脱は導入初期に集中している
- したがって: 機能差より、初期設定の摩擦が短期の成長制約である可能性が高い
- 意思決定では: 次の開発枠を新機能だけに使わず、オンボーディング改善案と比較する
- 確信度: Medium
- 反証: 新機能不足が失注理由になっているかは未確認

情報量を増やすのではなく、次の比較・判断が変わる形へ変換する。
